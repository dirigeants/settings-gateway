Protected parseTupleUpdateInput

  • parseTupleUpdateInput(changes: object | SettingsUpdateResults): SqlProviderParsedTupleUpdateInput
    • Defined in src/lib/structures/SQLProvider.ts:134

    Process the input from {@link Settings#update} or {@link Settings#reset} into an object with the keys and values that can be used for schema-based (SQL) database drivers. If it receives a non-array, it is flattened into a dotted object notation. Please note that this behaviour may be tricky when working with a SchemaEntry which type accepts an object and it's not an array, as it'll be flattened into as many keys as properties it has.

    Parameters

    • changes: object | SettingsUpdateResults

    Returns SqlProviderParsedTupleUpdateInput
